## 👨‍💻 Developed by **Idin Iskandar, S.Kom**
> **"Logic knows no hardware limits. Building high-end SaaS from a smartphone screen."** 🗿🔥

**Nexus Saver UI** adalah antarmuka modern dengan estetika *Cyberpunk/Midnight* yang dirancang khusus untuk memberikan pengalaman mengunduh video media sosial (TikTok, IG, FB, YT) yang premium dan bebas iklan mengganggu. 

Seluruh baris kode dalam proyek ini dikerjakan sepenuhnya (**100%**) melalui lingkungan **Termux Android** menggunakan perangkat smartphone.

---

## ✨ Fitur Unggulan

- 🎨 **Futuristic Dark UI** — Desain minimalis dengan efek neon glowing dan transisi halus menggunakan **Framer Motion**.
- 📱 **Mobile Optimized** — Responsif total untuk semua jenis layar browser ponsel, finger-friendly design.
- 🔍 **Auto-Detection System** — Cukup tempel link, dan biarkan sistem mengenali platform video secara otomatis.
- ⚡ **No Watermark Engine** — Terintegrasi dengan **Nexus Saver API** untuk mendapatkan link video asli tanpa tanda air.
- 📂 **Download History** — Riwayat unduhan tersimpan secara lokal dan aman di browser pengguna.

## 🚀 Local Installation

```bash
# Clone the repository
git clone https://github.com/idincodingweb/Nexus-Saver-UI.git

# Enter the directory
cd Nexus-Saver-UI

# Install dependencies (Jimat Termux)
npm install --legacy-peer-deps

# Run development server
npm run dev -- --host
