## Fitur
- UI modern & dark theme
- Deteksi otomatis platform
- Download cepat tanpa watermark
- Mobile friendly
- Riwayat unduhan (local storage)

## Tech Stack
- React + Vite + TypeScript
- Tailwind CSS + shadcn/ui
- Framer Motion
